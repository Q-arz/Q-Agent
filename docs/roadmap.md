## Roadmap

### 1. Robustness

- Absolute paths based on `__file__`
- Listener restart and error handling in IO

### 2. Security & Permissions

- Permission checks in router
- Time-bound elevation with audit logs

### 3. Module API v1

- Adopt `get_manifest()` with permissions and commands metadata
- Backward compatibility with `get_commands()`

### 4. Marketplace MVP

- Package format and checksum verification
- Installer and compatibility checks

### 5. System Control Module

- Safe operations: open apps, file ops, automation primitives

### 6. UI/UX

- System tray mini UI and notifications
